## Quick Reference — Expert Edition

| Tricky Distinction | Correct Understanding |
|-------------------|----------------------|
| **Phase A vs Prelim** | Phase A is first ADM cycle phase; Prelim sets up the capability |
| **Iteration types** | Cross-phase, within-phase, between-phase |
| **Conformant vs Consistent** | Conformant = all spec features present; Consistent = extra features added |
| **EC vs Repository** | EC = virtual classification; Repository = physical storage |
| **View vs Viewpoint** | View = representation; Viewpoint = rules for making the view |
| **Deliverable vs Artifact** | Deliverable = contractual; Artifact = granular work product |
| **SIB vs Ref Library** | SIB = mandatory standards; Ref Library = guidelines/templates |
| **ABB vs SBB** | ABB = logical/what; SBB = physical/how |
| **Contract vs SoW** | Contract = architecture↔implementation; SoW = architecture↔sponsor |
| **TRM vs III-RM** | TRM = platform services; III-RM = boundaryless information flow |
| **Strategic vs Capability** | Strategic = executive view; Capability = most detailed |
