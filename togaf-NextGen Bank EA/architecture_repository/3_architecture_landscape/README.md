# TOGAF Class: Architecture Landscape

The **Architecture Landscape** class contains active architectural views of the operating enterprise at different levels of granularity. It is divided into three tiers:

---

## 1. Strategic Architectures
*   **Path**: `strategic/`
*   **Contents**: High-level, long-term architectural views that establish the context and framework for segment and capability architectures. Primarily used for executive planning.

---

## 2. Segment Architectures
*   **Path**: `segment/`
*   **Contents**: Detailed, domain-specific designs that govern the development of specific business units or product lines.
*   **Active Segments**:
    - **[digital_lending_stp/](file:///Users/manavshrivastava/Documents/github/untitled%20folder/togaf/architecture_repository/3_architecture_landscape/segment/digital_lending_stp/README.md)**: The straight-through processing digital lending segment architecture for micro-loans.

---

## 3. Capability Architectures
*   **Path**: `capability/`
*   **Contents**: Detailed architectural blueprints for cross-cutting horizontal capabilities (e.g., enterprise IAM, shared transaction ledger platforms).
