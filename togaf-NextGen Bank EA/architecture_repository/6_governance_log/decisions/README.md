# Governance Log: Board Decisions

This directory contains the official records of all architecture decisions, framework approvals, and tech stack choices made by the **NextGen Bank Architecture Board**.

---

## 📋 Naming Convention
All files must follow the format `DEC-YYYY-XXXX_short_description.md`:
*   `DEC`: Prefix for decision.
*   `YYYY`: Year of the decision.
*   `XXXX`: Sequential 4-digit ID.

## 📐 Decision Record Template (Standard Structure)
Every decision file must contain:
1.  **Decision ID & Title**
2.  **Context & Problem Statement**: Why was this decision necessary?
3.  **Options Considered**: Table showing pros/cons.
4.  **Decision Outcome**: What option was selected and why?
5.  **Implications**: Downstream costs, code refactors, or resources needed.
6.  **Board Approvers**: List of voting members who approved the decision.
